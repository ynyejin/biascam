# BiasCam: Who’s Your Bias?

## 📌 프로젝트 소개

BiasCam은 단체 아이돌 퍼포먼스 영상에서 사용자가 선택한 멤버를 추적하여 개인 직캠을 생성하는 컴퓨터 비전 프로젝트입니다.

사용자는 영상 속에서 검출된 얼굴 중 자신의 “최애”를 선택할 수 있으며, 시스템은 OpenCV와 MediaPipe를 활용해 해당 멤버의 위치를 추적하고 자동으로 crop하여 개인 직캠 영상을 생성합니다.

본 프로젝트는 단체 영상 속 특정 인물을 선택하고 따라가는 과정을 통해 얼굴 검출, 객체 추적, 영상 crop, 영상 저장 등 OpenCV 기반 영상 처리 흐름을 구현하는 것을 목표로 합니다.

---

## 🎯 주요 기능

- 단체 퍼포먼스 영상 입력
- MediaPipe 기반 얼굴 검출
- 사용자가 원하는 멤버 얼굴 선택
- 선택한 멤버의 위치 추적
- bbox smoothing을 통한 화면 흔들림 완화
- 선택 멤버 중심 자동 crop
- 개인 직캠 영상 저장

---

## 🛠 사용 기술

- Python
- OpenCV
- MediaPipe
- NumPy

---

## 📂 프로젝트 목표

단체 영상에서 특정 인물을 선택하고, 해당 인물을 중심으로 자동 직캠을 생성하는 컴퓨터 비전 기반 영상 처리 시스템을 구현하는 것을 목표로 합니다.
